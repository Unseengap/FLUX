# Phase 10 Specification: Hybrid Wave+Byte Universal Generation
## The Best of Both Worlds — Precision When Needed, Speed When Possible

> Prerequisites: `Flux-beta.flx` must exist (contains all Phases 1-8 components).
> Copilot: Open SPECIFICATION.md + PHASE_BETA_SPEC.md + flux-domanant-domains.md + this file.
>
> **Phase 8 is NOT legacy.** It is the trained foundation in `Flux-beta.flx`.
> Phase 10 EXTENDS the `.flx` model with Phase 9's wave-level generation,
> creating a hybrid that can use BOTH generation paths based on the task.

---

## The Core Insight: Why Hybrid?

Phase 8 (byte-level) and Phase 9 (wave-level) each have strengths:

| Aspect | Phase 8 (ByteDecoder) | Phase 9 (WaveGenerator) |
|--------|----------------------|------------------------|
| **Precision** | Character-perfect (code, formulas) | Word-approximate |
| **Speed** | ~100 steps/sentence | ~15 steps/sentence |
| **Coherence** | Can garble between words | Thinks in concepts |
| **Modality** | Text-only | Universal (swap last-mile) |
| **Training** | Trained (loss 1.8, ppl ~6) | Needs more training |
| **Use case** | Exact strings, rare words | Chat, summaries, fast |

**The hybrid answer:** Use wave-level for semantic planning, byte-level for precision.
Route automatically based on task requirements.

## Goal

Build a unified hybrid model that:

1. **Loads entirely from `Flux-beta.flx`** — all Phase 1-8 components
2. **Adds Phase 9 wave modules** as a parallel generation path
3. **Routes intelligently** between wave-mode and byte-mode
4. **Preserves all FLUX advantages** — zero forgetting, one-shot, O(log n)
5. **Enables multimodal extension** — swap WaveToText for WaveToImage/Audio/Mol

        'version': version,
        'metadata': flx.get('metadata', {}),

def save_flx(
    path: Path,
    components: Dict[str, Any],
    version: str = '1.1-hybrid',
    metadata: Optional[Dict] = None,
) -> None:
    """
    Save components to .flx format.
    
    Preserves modular structure for component-level upgrades.
    """
    flx = {
        'format': 'FLUX',
        'version': version,
        'metadata': metadata or {},
    }
    
    # Serialize each component
    for name, module in components.items():
        if module is None:
            flx[name] = None
        elif hasattr(module, 'state_dict'):
            flx[name] = {
                'config': getattr(module, 'config', {}),
                'state_dict': module.state_dict(),
            }
        else:
            flx[name] = module  # raw dict (e.g., memory)
    
    torch.save(flx, path)
    size_mb = path.stat().st_size / (1024 * 1024)
    print(f"  ✓ Saved {path.name} ({size_mb:.1f} MB)")

## Component 1: FLUXHybrid — The Unified Model

```python
class FLUXHybrid(nn.Module):
    """
    Unified FLUX model with dual generation paths.
    
    Loads from Flux-beta.flx (v1.0-beta or v1.1-hybrid) and provides:
    - Byte-mode generation via Phase 8 WaveDecoder (precise)
    - Wave-mode generation via Phase 9 modules (fast, semantic)
    - Automatic routing based on task characteristics
    - Extension points for multimodal outputs
    
    The core (CSE, Field, GR, TL, CGN, Memory) is ALWAYS from .flx.
    Generation paths are pluggable.
    
    Args:
        flx_path: Path to Flux-beta.flx file
        device: Target device
    """
    
    def __init__(
        self,
        flx_path: Path = Path('checkpoints/Flux-beta.flx'),
        device: str = 'cpu',
    ):
        super().__init__()
        
        # Load everything from .flx
        loaded = load_flx(flx_path, device)
        self.version = loaded['version']
        self.metadata = loaded['metadata']
        
        components = loaded['components']
        
        # ── Core components (from .flx, always present) ──
        self.cse = components['cse']
        self.field = components['field']
        self.memory = components['memory']
        
        # ── Byte-mode: Phase 8 WaveDecoder (from .flx) ──
        self.byte_decoder = components['decoder']
        
        # ── Wave-mode: Phase 9 modules (from .flx or init fresh) ──
        self.wave_generator = components.get('wave_generator')
        self.wave_chunker = components.get('wave_chunker')
        self.wave_to_text = components.get('wave_to_text')
        
        # ── Router (from .flx or init fresh) ──
        self.task_router = components.get('task_router') or TaskRouter()
        
        # ── Multimodal extensions (None until added) ──
        self.wave_to_image = components.get('wave_to_image')
        self.wave_to_audio = components.get('wave_to_audio')
        self.wave_to_mol = components.get('wave_to_mol')
        
        self._device_str = device
        self._wave_mode_available = all([
            self.wave_generator,
            self.wave_chunker,
            self.wave_to_text,
        ])
        
        print(f"  ✓ FLUXHybrid loaded from {flx_path.name}")
        print(f"    Byte-mode: ✓ (Phase 8 WaveDecoder)")
        print(f"    Wave-mode: {'✓' if self._wave_mode_available else '✗ (not trained)'}")
    
    @classmethod
    def from_flx(cls, path: str = 'checkpoints/Flux-beta.flx', device: str = 'cpu'):
        """Load FLUXHybrid from .flx file."""
        return cls(flx_path=Path(path), device=device)
    
    def generate(
        self,
        prompt: str,
        max_length: int = 100,
        mode: str = 'auto',
        temperature: float = 0.8,
        output_modality: str = 'text',
    ) -> str:
        """
        Generate output from prompt.
        
        Args:
            prompt: Input text
            max_length: Maximum output length
            mode: 'auto', 'wave' (fast), or 'byte' (precise)
            temperature: Sampling temperature
            output_modality: 'text', 'image', 'audio', 'mol'
        
        Returns:
            Generated output (str for text, tensor for other modalities)
        """
        # Encode prompt → field context
        wave_seq = self.cse.encode(prompt).full.to(self._device_str)
        wave_vec = wave_seq.mean(dim=0)
        field_features, _, _ = self.field.query(wave_vec, k=4)
        context = field_features.mean(dim=0)
        
        # Route to appropriate generation mode
        if mode == 'auto':
            mode = self.task_router.route(prompt, output_modality)
        
        if mode == 'wave' and self._wave_mode_available:
            return self._generate_wave_mode(
                prompt, context, max_length, temperature, output_modality
            )
        else:
            return self._generate_byte_mode(
                prompt, context, max_length, temperature
            )
    
    def _generate_wave_mode(
        self,
        prompt: str,
        context: torch.Tensor,
        max_length: int,
        temperature: float,
        output_modality: str,
    ) -> str:
        """Fast wave-level generation."""
        # Generate waves (thinking in concepts)
        max_waves = max_length // 5  # ~5 chars per word average
        generated_waves, confidences = self.wave_generator.generate(
            field_context=context,
            max_waves=max_waves,
        )
        
        # Route to appropriate last-mile decoder
        if output_modality == 'text':
            return self._waves_to_text(generated_waves, temperature)
        elif output_modality == 'image' and self.wave_to_image:
            return self.wave_to_image(generated_waves)
        elif output_modality == 'audio' and self.wave_to_audio:
            return self.wave_to_audio(generated_waves)
        elif output_modality == 'mol' and self.wave_to_mol:
            return self.wave_to_mol(generated_waves)
        else:
            # Fallback to text
            return self._waves_to_text(generated_waves, temperature)
    
    def _waves_to_text(
        self,
        waves: torch.Tensor,
        temperature: float,
    ) -> str:
        """Convert generated waves to text via WaveToText."""
        text_parts = []
        for wave in waves:
            chunk_bytes = self.wave_to_text.decode(wave, temperature=temperature)
            text_parts.append(chunk_bytes.decode('utf-8', errors='replace'))
        return ' '.join(text_parts)
    
    def _generate_byte_mode(
        self,
        prompt: str,
        context: torch.Tensor,
        max_length: int,
        temperature: float,
    ) -> str:
        """Precise byte-level generation via Phase 8 WaveDecoder."""
        return self.byte_decoder.generate(
            context=context,
            max_bytes=max_length,
            temperature=temperature,
        )
    
    def learn_fact(self, fact: str) -> None:
        """One-shot fact learning via episodic memory (unchanged)."""
        wave_seq = self.cse.encode(fact).full.to(self._device_str)
        wave_vec = wave_seq.mean(dim=0)
        self.memory['episodic'].add(wave_vec, metadata={'fact': fact})
    
    def query(self, question: str) -> str:
        """Query memory and generate answer."""
        wave_seq = self.cse.encode(question).full.to(self._device_str)
        wave_vec = wave_seq.mean(dim=0)
        
        # Search episodic memory
        results = self.memory['episodic'].search(wave_vec, k=3)
        if results:
            # Use retrieved facts as context
            return self.generate(question + '\n' + results[0]['fact'])
        return self.generate(question)
    
    def save_flx(self, path: Path) -> None:
        """Save current state to .flx file."""
        save_flx(
            path=path,
            components={
                'cse': self.cse,
                'field': self.field,
                'memory': self.memory,
                'decoder': self.byte_decoder,
                'wave_generator': self.wave_generator,
                'wave_chunker': self.wave_chunker,
                'wave_to_text': self.wave_to_text,
                'task_router': self.task_router,
                'wave_to_image': self.wave_to_image,
                'wave_to_audio': self.wave_to_audio,
                'wave_to_mol': self.wave_to_mol,
            },
            version='1.1-hybrid',
            metadata={
                'upgraded_from': self.version,
                'learning_steps': getattr(self, '_learning_steps', 0),
            },
        )
```

## Component 2: TaskRouter — Intelligent Mode Selection

```python
class TaskRouter(nn.Module):
    """
    Route generation requests to appropriate mode.
    
    Decisions based on:
    - Task type (chat vs code vs exact)
    - Output modality (text vs image vs audio)
    - Prompt characteristics (length, technical markers)
    - User preference override
    
    Modes:
    - 'wave': Fast, semantic, good for chat/summaries
    - 'byte': Precise, character-perfect, good for code/formulas
    """
    
    # Patterns that suggest byte mode (precision needed)
    BYTE_PATTERNS = [
        r'```',              # Code blocks
        r'def |class |import ', # Python code
        r'\$.*\$',           # LaTeX math
        r'http[s]?://',      # URLs
        r'@\w+|#\w+',        # Mentions/hashtags
        r'\b[A-Z]{2,}\b',    # Acronyms
        r'\d+\.\d+',         # Version numbers
    ]
    
    # Patterns that suggest wave mode (semantic understanding)
    WAVE_PATTERNS = [
        r'explain|describe|summarize',
        r'what is|how does|why',
        r'tell me about|help me understand',
        r'^chat|^talk|^discuss',
    ]
    
    def __init__(self, default_mode: str = 'wave'):
        super().__init__()
        self.default_mode = default_mode
        self._byte_re = re.compile('|'.join(self.BYTE_PATTERNS), re.I)
        self._wave_re = re.compile('|'.join(self.WAVE_PATTERNS), re.I)
    
    def route(
        self,
        prompt: str,
        output_modality: str = 'text',
    ) -> str:
        """
        Decide generation mode for this request.
        
        Returns: 'wave' or 'byte'
        """
        # Non-text output → always wave mode (byte mode is text-only)
        if output_modality != 'text':
            return 'wave'
        
        # Check for precision-requiring patterns
        if self._byte_re.search(prompt):
            return 'byte'
        
        # Check for semantic-understanding patterns
        if self._wave_re.search(prompt):
            return 'wave'
        
        # Default based on prompt length
        # Short prompts → byte (often exact queries)
        # Long prompts → wave (likely discussion)
        return 'wave' if len(prompt) > 100 else self.default_mode
    
    def route_with_reason(self, prompt: str, output_modality: str = 'text'):
        """Route and explain why."""
        mode = self.route(prompt, output_modality)
        
        if output_modality != 'text':
            reason = f"Non-text output ({output_modality}) requires wave mode"
        elif mode == 'byte' and self._byte_re.search(prompt):
            match = self._byte_re.search(prompt).group()
            reason = f"Detected precision pattern: '{match}'"
        elif mode == 'wave' and self._wave_re.search(prompt):
            match = self._wave_re.search(prompt).group()
            reason = f"Detected semantic pattern: '{match}'"
        else:
            reason = f"Default ({self.default_mode}) based on prompt length"
        
        return mode, reason

## Component 3: Training the Wave Modules on .flx Base

```python
class HybridTrainer:
    """
    Train Phase 9 wave modules while keeping Phase 8 decoder frozen.
    
    Strategy:
    1. Load Flux-beta.flx (all Phase 1-8 components)
    2. Initialize fresh wave modules (WaveGenerator, WaveChunker, WaveToText)
    3. Freeze: CSE, Field, ByteDecoder (Phase 8)
    4. Train: WaveGenerator, WaveChunker, WaveToText
    5. Save as .flx v1.1-hybrid (preserving all original components)
    
    The ByteDecoder stays frozen — it's already trained and provides
    the precision fallback. Wave modules learn to match its output
    while being faster.
    """
    
    def __init__(
        self,
        flx_path: Path = Path('checkpoints/Flux-beta.flx'),
        device: str = 'cpu',
        lr: float = 3e-4,
    ):
        self.device = device
        
        # Load everything from .flx
        self.model = FLUXHybrid.from_flx(str(flx_path), device)
        
        # Freeze base components (already trained)
        for param in self.model.cse.parameters():
            param.requires_grad = False
        for param in self.model.field.parameters():
            param.requires_grad = False
        for param in self.model.byte_decoder.parameters():
            param.requires_grad = False
        
        # Initialize fresh wave modules if not present
        if not self.model._wave_mode_available:
            print("  ℹ Initializing fresh wave modules...")
            self._init_wave_modules()
        
        # Optimizer for wave modules only
        wave_params = (
            list(self.model.wave_generator.parameters()) +
            list(self.model.wave_chunker.parameters()) +
            list(self.model.wave_to_text.parameters())
        )
        self.optimizer = torch.optim.AdamW(wave_params, lr=lr)
    
    def _init_wave_modules(self):
        """Initialize fresh wave modules."""
        field_features = self.model.field.config.get('features', 512)
        
        self.model.wave_generator = WaveGenerator(
            wave_dim=432,
            field_features=field_features,
        ).to(self.device)
        
        self.model.wave_chunker = WaveChunker(
            wave_dim=432,
        ).to(self.device)
        
        self.model.wave_to_text = WaveToText(
            wave_dim=432,
            hidden_dim=256,
            max_bytes=20,
        ).to(self.device)
        
        self.model._wave_mode_available = True
    
    def train_step(self, text: str) -> dict:
        """
        Single training step.
        
        Uses ByteDecoder output as soft target — wave modules learn
        to produce similar output faster.
        """
        # Encode text → waves
        wave_seq = self.model.cse.encode(text).full.to(self.device)
        wave_vec = wave_seq.mean(dim=0)
        
        # Get field context (frozen)
        with torch.no_grad():
            field_features, _, _ = self.model.field.query(wave_vec, k=4)
            context = field_features.mean(dim=0)
        
        # Chunk into word-level targets
        target_chunks, spans = self.model.wave_chunker(wave_seq)
        if target_chunks.shape[0] < 3:
            return {'loss': 0.0, 'skipped': True}
        
        # WaveGenerator predicts next chunk from context + previous
        predicted_waves = self.model.wave_generator.forward_teacher(
            field_context=context,
            target_waves=target_chunks,
        )
        
        # Loss 1: Wave prediction (MSE + cosine)
        target = target_chunks[1:]
        mse_loss = F.mse_loss(predicted_waves, target)
        cos_loss = 1.0 - F.cosine_similarity(predicted_waves, target, dim=-1).mean()
        wave_loss = mse_loss + cos_loss
        
        # Loss 2: Spelling via WaveToText
        text_bytes = list(text.encode('utf-8', errors='replace'))
        spelling_loss = torch.tensor(0.0, device=self.device)
        n_chunks = 0
        
        for pred_wave, (start, end) in zip(predicted_waves, spans[1:]):
            actual = text_bytes[start:end]
            if not actual or len(actual) > 20:
                continue
            byte_target = torch.tensor(actual, dtype=torch.long, device=self.device)
            logits = self.model.wave_to_text(pred_wave, byte_target)
            chunk_loss = F.cross_entropy(logits[:len(byte_target)], byte_target)
            spelling_loss = spelling_loss + chunk_loss
            n_chunks += 1
        
        if n_chunks > 0:
            spelling_loss = spelling_loss / n_chunks
        
        # Total loss
        total_loss = wave_loss + 0.5 * spelling_loss
        
        # Backward
        self.optimizer.zero_grad()
        total_loss.backward()
        torch.nn.utils.clip_grad_norm_(
            list(self.model.wave_generator.parameters()) +
            list(self.model.wave_to_text.parameters()),
            max_norm=1.0,
        )
        self.optimizer.step()
        
        return {
            'loss': total_loss.item(),
            'wave_loss': wave_loss.item(),
            'spelling_loss': spelling_loss.item(),
            'n_chunks': target_chunks.shape[0],
        }
```

class WaveToX(nn.Module):
    """
    Base class for wave → modality decoders.
    
    All decoders share the same interface:
    - Input: [N, 432] wave sequence from WaveGenerator
    - Output: Modality-specific tensor
    
    The WaveGenerator is TOTALLY modality-agnostic. It produces
    semantic waves. The last-mile decoder interprets them.

        self.wave_dim = wave_dim
    
    def decode(self, waves: torch.Tensor) -> Any:

class WaveToImage(WaveToX):
    """
    Wave → Image tensor.
    
    Architecture: waves → latent grid → diffusion decoder → pixels
    
    Each wave maps to a region of the image. The wave's semantic
    content determines what appears in that region.
    """
    
    def __init__(
        self,
        wave_dim: int = 432,
        image_size: int = 256,
        latent_dim: int = 4,  # For diffusion latent space
    ):
        super().__init__(wave_dim)
        self.image_size = image_size
        
        # Project waves to latent patches
        self.wave_to_latent = nn.Linear(wave_dim, latent_dim * 16 * 16)
        
        # Simple decoder (placeholder — would use actual diffusion)
        self.decoder = nn.Sequential(
            nn.ConvTranspose2d(latent_dim, 128, 4, 2, 1),
            nn.ReLU(),
            nn.ConvTranspose2d(128, 64, 4, 2, 1),
            nn.ReLU(),
            nn.ConvTranspose2d(64, 3, 4, 2, 1),
            nn.Sigmoid(),
        )
    
    def decode(self, waves: torch.Tensor) -> torch.Tensor:
        """
        Decode wave sequence to image.
        
        Args:
            waves: [N, 432] semantic waves
        
        Returns:
            [3, H, W] RGB image tensor
        """
        # Average waves (or arrange spatially for larger images)
        wave_avg = waves.mean(dim=0)
        
        # Project to latent
        latent = self.wave_to_latent(wave_avg)
        latent = latent.view(1, -1, 16, 16)
        
        # Decode to image
        image = self.decoder(latent)
        return image.squeeze(0)


class WaveToAudio(WaveToX):
    """
    Wave → Audio waveform.
    
    Architecture: waves → mel spectrogram → vocoder → audio samples
    
    Each wave maps to a time slice of audio. The wave's content
    determines the acoustic features of that slice.
    """
    
    def __init__(
        self,
        wave_dim: int = 432,
        n_mels: int = 80,
        hop_length: int = 256,
    ):
        super().__init__(wave_dim)
        self.n_mels = n_mels
        self.hop_length = hop_length
        
        # Project waves to mel frames
        self.wave_to_mel = nn.Linear(wave_dim, n_mels)
        
        # Simple vocoder (placeholder — would use HiFi-GAN etc.)
        self.vocoder = nn.Sequential(
            nn.ConvTranspose1d(n_mels, 256, 8, 4, 2),
            nn.LeakyReLU(0.2),
            nn.ConvTranspose1d(256, 1, 8, 4, 2),
            nn.Tanh(),
        )
    
    def decode(self, waves: torch.Tensor) -> torch.Tensor:
        """
        Decode wave sequence to audio.
        
        Args:
            waves: [N, 432] semantic waves
        
        Returns:
            [samples] audio waveform tensor
        """
        # Each wave → one mel frame
        mel_frames = self.wave_to_mel(waves)  # [N, 80]
        mel = mel_frames.T.unsqueeze(0)  # [1, 80, N]
        
        # Vocoder to audio
        audio = self.vocoder(mel)
        return audio.squeeze()
